- 1、大盘反转的信号，反转时整个市场的结构，反推哪些信号出来大盘就要反转，事实上量价是最有效的，但是不同时期使用不同周期的量价来判断
- 2、反转前中后期各板块的涨幅
- 3、每个板块的龙头，如何判断出龙头
- 4、龙头的回调方式
- 5、选中的龙头进行组合建仓，分批建仓的方式怎么保证安全
- 6、出场逻辑
- 7、不同的标的，不同周期多空指标不一，要搞明白每只标的独有的多空判断规则，比如月D从高位跌破50看空
- 8、右侧15分钟恐慌点低吸
- 9、计算概率，比如右侧量价突破后，在一定时间之内盈利的概率以及最多承受浮亏的空间和时间
